# Nimora

**Nimora** - гибкая система настольной ролевой игры, ориентированная на творческое решение проблем, игру вокруг персонажей и быстрое принятие решений, а не на строгое следование правилам.
В ней используется простая система на основе d20, ранжированные черты персонажей, гибкие способности, масштабируемые предметы, система истощения и подробные рекомендации для ведущего.

## Цели разработки

- Поощрять игроков решать проблемы творчески, а не повторять одни и те же схемы из игры в игру.
- Сделать механику легко адаптируемой к разным сеттингам.
- Уменьшить влияние чисел на темп игры.
- Дать ведущему достаточно пространства для творчества и повествования.

## Статус

Система была создана в основном для обучения и игры с друзьями. Тем не менее, в нее уже можно играть, и я надеюсь, что она поможет кому-нибудь хорошо провести время.
